# TalkBetter.ai 需求管理文档

## 项目背景
TalkBetter.ai 是一款专注普通话发音训练的 Web 应用。通过“听示范 -> 模仿读 -> 实时评测 -> 针对性纠错”的闭环，帮助用户纠正字音，提升口语表达能力。

## 功能清单 (Features)

### F-001: 基础 UI 框架 (已完成)
- **US-001**: 响应式训练界面，支持显示汉字、拼音和难度标签。
- **US-002**: 训练进度条和状态反馈（如“正在聆听”、“发现错误”）。

### F-002: 语音播放与示范 (已完成)
- **US-003**: 电脑范读功能（支持 Web Speech API 和 Capacitor Native TTS）。
- **US-004**: 点击单个汉字重听示范发音。

### F-003: 实时发音评测 (开发中)
- **US-005**: 采集用户录音并转换为文字。
- **US-006**: 基于拼音的精准比对算法，识别发音错误（已在 `speech-utils.ts` 实现）。
- **AC-001**: 必须能准确识别出声调或拼音错误的汉字并标记为红色。

### F-004: 循环训练逻辑 (已完成)
- **US-007**: 完整训练流程：范读 -> 尝试读 -> 纠错读 -> 全文复述。
- **US-008**: 纠错阶段仅需针对错误字词进行复读。

### F-005: 跨平台适配 (优化中)
- **US-009**: 支持普通浏览器环境运行。
- **US-010**: 支持 Android 原生环境运行（Capacitor）。
- **AC-002**: 解决浏览器环境下麦克风权限和识别不稳定问题。

## 验收标准 (Acceptance Criteria)
- [x] 页面能正常渲染练习句子。
- [x] 点击“听示范”能听到声音且有文字高亮。
- [x] 点击单个汉字能听到对应发音。
- [ ] 录音识别在 Chrome 和 Android WebView 中均能稳定工作。
- [x] 评测算法能正确区分对错。

## 当前进度
- **已完成**: UI 交互、TTS 示范、拼音评测逻辑、训练状态机。
- **进行中**: 优化浏览器/WebView 环境下的语音识别 (Speech Recognition) 稳定性。
